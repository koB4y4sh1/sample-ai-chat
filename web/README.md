# @zenith/web

Next.js frontend and BFF workspace package for Zenith AI Chat.

The App Router endpoint at `/api/copilotkit` hosts CopilotKit Runtime and
connects to the AG-UI service configured by `AG_UI_BASE_URL`.

## Commands

```bash
pnpm --filter @zenith/web dev
pnpm --filter @zenith/web build
pnpm --filter @zenith/web typecheck
```
