Zenith AI Chat 開発環境仕様

作成日: 2026-04-24 / 対象: モノレポ初期開発環境

目的

FE/BFF、AGENT、MCPを1つのリポジトリで管理し、開発開始時点からformat、lint、 type check、pre-commitを同じ入口で実行できる状態にする。

構成

領域 ディレクトリ 技術 責務
FE/BFF web TypeScript, Next.js, React, CopilotKit, pnpm チャットUI、CopilotKit Runtime、agentへのBFF route handler。
BE agent Python, AgentFramework, uv エージェント実行とagent endpointの公開。
MCP mcp Python, FastMCP, uv エージェントから利用するツール群のMCP公開。

標準コマンド

目的 コマンド
JavaScript依存関係の同期 pnpm install
Python依存関係の同期 uv sync --all-packages --dev
agent開発サーバー pnpm run dev:agent
Next.js FE/BFF開発サーバー pnpm run dev:web
整形 pnpm run format
lint pnpm run lint
型検査 pnpm run typecheck
品質ゲート一括実行 pnpm run check
pre-commit有効化 uv run pre-commit install

接続仕様

  • webはNext.js App RouterでUIとBFF route handlerを管理する。
  • チャットUIは/api/copilotkitのCopilotKit Runtimeへ接続する。
  • Next.js BFF routeはAG_UI_BASE_URLの/copilotkitへagent通信を転送する。
  • agentはsrc/main.pyでMicrosoft Agent Frameworkのagent endpointを公開する。

品質ゲート

TypeScript

  • biome.jsonでformat、lint、import整理を統一する。
  • webはtsc --noEmitで型検査する。
  • npm lockfileは使用せず、pnpm workspaceのルートロックに集約する。

Python

  • uv workspaceでagentとmcpを管理する。
  • ruffでlintとformatを実行する。
  • mypy strictで型検査する。
  • bandit設定はpyproject.tomlに集約する。

Git Hooks

  • pre-commitは基本検査、uv lock、Biome、Ruff、mypyを実行する。
  • hookはローカル環境のpnpmとuvを使用する。

環境変数

変数 配置 用途
AG_UI_BASE_URL web/.env Next.js BFF routeが接続するagent serviceのbase URL。
ZENITH_OPENAI_MODEL agent/.env agent serviceが使用するOpenAI model名。
